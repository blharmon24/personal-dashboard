🔖 Athletic.net Sync Bookmarklet

A one-click tool to sync your kids' athletic.net results into the dashboard.

Step 1 — Add the bookmarklet to your bookmarks bar

1

First, make sure your bookmarks bar is visible. In Chrome: press Ctrl+Shift+B to toggle it on.

2

Drag the button below up to your bookmarks bar and drop it there. It will appear as a bookmark called "Sync → Supabase".

Sync → Supabase

☝️ Drag this button to your bookmarks bar

3

You should now see "Sync → Supabase" in your bookmarks bar.

Step 2 — One more thing: allow the bookmarklet to write to Supabase

Run this SQL in the Supabase SQL editor to allow the bookmarklet to save results:

CREATE POLICY "allow anon insert" ON kids_results FOR INSERT TO anon WITH CHECK (true); CREATE POLICY "allow anon update" ON kids_results FOR UPDATE TO anon USING (true);

How to use it

1. Go to athletic.net and make sure you're logged in
2. Navigate to Tanner's page: athletic.net/athlete/21290627/track-and-field/all
3. Click "Sync → Supabase" in your bookmarks bar — a green confirmation will appear on the page
4. Navigate to Luke's page: athletic.net/athlete/22163804/track-and-field/all
5. Click "Sync → Supabase" again
6. Go back to Kids Running and refresh — results will be there
